# Telegram Intake Architecture

Telegram intake для сделок и документов отключён.

Активный Telegram-бот сейчас поддерживает только:

- привязку пользователя через `/start <код>`;
- справку через `/help`;
- исходящие уведомления CRM по настройкам пользователя.

Исторические модели `TelegramInboundMessage` и `TelegramDealRoutingSession` остаются в
БД и админке только для просмотра старых данных. Новые сообщения, файлы, callback-кнопки
и команды выбора/создания сделок бот больше не обрабатывает как intake-пакеты.
